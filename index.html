PoPi

C++ Qt LGPL v3.0

PoPi is an application designed as a screenshot tool for the GNU/Linux operating system under the X11 windowing system. The application takes a screenshot using a hotkey, uploads the image to Imgur, and pastes the link onto the clipboard.
