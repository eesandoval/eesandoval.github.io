PoPi - The Screenshot Tool

PoPi is an application designed as a screenshot tool for the GNU/Linux operating system. This application targets the X11 windowing system and was initially developed for KDE Plasma, but may work on any other X11 windowing system regardless of desktop environment. Unlike many other screenshot tools available on GNU/Linux, this application is designed to mimic similar functions to popular Window applications puush and ShareX. These applications take a screenshot, save the screenshot image locally, upload the image to an image hosting site (such as Imgur), and finally copy the URL where the image was uploaded to onto the clipboard. This can all easily be done in one simple hotkey motion on Windows using the aformentioned programs. Similar functionallity could be obtained in GNU/Linux using other screenshot tools, however it was not as efficient as one simple hotkey and often involved multiple button clicks and user interfaces. PoPi is designed to be simple enough to use and provide an easy way to auto-upload images to share them with others.
